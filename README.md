# Мастер Потолков

Портфолио-лендинг для вымышленного мастера по натяжным потолкам. Проект собран как отдельный Astro-сайт в собственной папке и подходит для демонстрации фронтенд-работы в портфолио.

## Стек

- Astro
- Tailwind CSS
- HTML, CSS, JavaScript

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
